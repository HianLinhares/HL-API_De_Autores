                                    ---------------  MÉTODO 1 ---------------

Para versionar a versão para um deploy a meneira mais fácil de realizar esse processo é criando
um arquivo application-production ou application-homologacao e no application principal utilizado para
as configurações realizar as seguintes definições:


spring:
  application:
    name: HLSpringBoot
  profiles:
    active: homologacao

 onde o active vai referenciar o application-xxx.yml que será utilizado


                                     ---------------  MÉTODO 2 ---------------

No próprio arquivo application.yml separar por 3 traços (---) e realizar as configurações do ambiente

# Configurações de produção
---
spring:
  config:
    activate:
      on-profile: production
  jpa:
    show-sql: false

# Configurações de homologação
---
spring:
  config:
    activate:
      on-profile: homologacao
  jpa:
    show-sql: true
